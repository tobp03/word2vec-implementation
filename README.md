# word2vec-implementation

Word2Vec Skip-Gram with Negative Sampling (SGNS) implementation using NumPy.

The notebook covers:
- text preprocessing
- skip-gram pair generation
- model training with negative sampling
- cosine similarity evaluation of learned embeddings

Everything is implemented **from scratch in a single notebook** for educational purposes.

More explanations **[here](https://tobypurbojo.com/notes/Understanding%20Word2Vec%20324df4fe-71d8-80a3-bf8d-f09ff99a71c0)**
